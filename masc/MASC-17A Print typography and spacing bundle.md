# MASC-17A Print typography and spacing bundle

## Purpose
- Handle the print-typography and spacing fixes from the processed MASC intake.

## Scope
- This bundle covers typography, bullets, and spacing behavior in the factsheet print experience.
- Target size: `5h`

## Checklist
- [ ] Indent the numbered bullets.
- [ ] Increase the secondary page title size on the print stylesheet so it reads clearly as a title.
- [ ] Reduce bullet size.
- [ ] Reduce the gap between bullet and text.
- [ ] Reduce the vertical spacing between bullet points.
- [ ] Normalize page-header spacing across print pages.
- [ ] Add more footer spacing in print.
- [ ] Add spacing between the first and second section headings, including `Purpose` to `Eligibility`, and apply globally if possible.

## Dependencies
- No explicit external blocker.
- If typography changes widen into design adjustments, stop and re-scope.

## Definition of Done
- Print output is more compact and readable.
- Bullet, title, header, and footer spacing changes are visible on affected factsheets.
- The changes are applied consistently enough to count as a shared print pass, not a one-page patch.
