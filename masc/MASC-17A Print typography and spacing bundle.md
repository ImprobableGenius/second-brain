- [x] Reduce bullet size.

- [x] Normalize page-header spacing across print pages.
- [x] Add more footer spacing in print.

- No explicit external blocker.
- If typography changes widen into design adjustments, stop and re-scope.

## Definition of Done
- Print output is more compact and readable.
- Bullet, title, header, and footer spacing changes are visible on affected factsheets.
- The changes are applied consistently enough to count as a shared print pass, not a one-page patch.
